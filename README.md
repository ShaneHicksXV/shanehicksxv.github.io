West Texas Substation Project

This repository presents a simplified project package for building a new electrical substation in West Texas. It combines project management documentation with a two-page informational website that could be used to introduce the project to stakeholders and the local community.

## Live Site

- Live Demo: https://github.com/ShaneHicksXV/shanehicksxv.github.io.git

## Documentation

All project management documents are located in the `docs/` folder:

- [Project Scope Statement](./docs/scope.md)
- [Project Plan & Risk Analysis](./docs/plan.md)
- [Project Retrospective](./docs/retrospective.md)

## Website Files

- `index.html` – Project Overview
- `about.html` – Additional project details
- `style.css` – Shared styling for both pages

## Purpose

This repository is a demonstration of hybrid generalist thinking and unofficial project management for a small, well-defined technical project: planning and communicating the construction of a new substation in West Texas.
